# FastAPI Docker Application

## Overview
This FastAPI Docker application is designed to provide a simple template for deploying FastAPI services using Docker containers. It demonstrates how to create, build, and run a FastAPI application within a Docker environment.

## Features
- **FastAPI Framework**: Built on the modern and fast (high-performance) web framework for building APIs with Python 3.7+.
- **Docker Support**: Containerized application allowing easy deployment and scaling.
- **RESTful API**: Demonstrates RESTful API principles.
- **Automatic Interactive API Documentation**: Provides autogenerated documentation using Swagger UI and ReDoc.

## Prerequisites
- [Docker](https://www.docker.com/get-started) installed on your machine.
- Basic knowledge of Python and REST APIs.

## Getting Started

### Clone the Repository
```bash
git clone https://github.com/githubgobi/fastapi-first-docker-app.git
cd fastapi-first-docker-app
```

### Build the Docker Image
```bash
docker build -t fastapi-first-docker-app .
```

### Run the Docker Container
```bash
docker run -d --name fastapi-app -p 8000:8000 fastapi-first-docker-app
```

### Access the Application
Open your browser and go to `http://localhost:8000/docs` to view the interactive API documentation.
